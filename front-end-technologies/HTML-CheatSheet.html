HTML Boilerplate
The fundamental structure that every HTML page should have. It includes the `DOCTYPE`, `head`, and `body`. 
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
   </head>
  <body>
     <!-- Content goes here -->
    </body>
</html>
 This code defines the basic structure that is required for a valid HTML document.
It includes the document type declaration, the root HTML element, and the head and body sections.

Meta Tags
Provide metadata about the HTML document, such as character set and viewport settings.
  • `<meta charset="UTF-8">` - Sets the character encoding.
  • `<meta name="viewport" content="width=device-width, initial-scale=1.0">` - Ensures responsive design.
 
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
 These tags help ensure proper rendering and responsiveness of the webpage across different devices.
Title Tag
Sets the title of the webpage, which appears in the browser tab. 
<title>Page Title</title>
 The title is displayed in the browser's title bar or tab, and is also used by search engines.

Headings Tags
Used to define headings in a document, from `<h1>` (most important) to `<h6>` (least important). 
<!DOCTYPE html>
<html>
    <head>
        <title>Headings Tags</title>
    </head>
    <body>
       <h1>Heading 1</h1>
       <h2>Heading 2</h2>
      <h3>Heading 3</h3>
      <h4>Heading 4</h4>
      <h5>Heading 5</h5>
      <h6>Heading 6</h6>
    </body>
</html>

Text formating Tags
Tags for formatting text within paragraphs.
  • `<strong>` - Bold text.
  • `<em>` - Italic text.
  • `<u>` - Underlined text.
  • `<mark>` - Highlighted text.
  • `<small>` - Smaller text.
  • `<`
  • `<del>` - Deleted text.
  • `<ins>` - Inserted text.
  • `<sub>` - Subscript text.
  • `<sup>` - Superscript text.

Lists
Used to group related items, either in order or without.
  • Unordered lists (`<ul>`) use bullet points.
  • Ordered lists (`<ol>`) use numbers.
  • List items are defined with the `<li>` tag.
 
<ul>
    <li>Coffee</li>
    <li>Tea</li>
</ul>
<ol>
    <li>Step 1</li>
    <li>Step 2</li>
</ol>

  • Tea
  1. Step 1
