Introduction to HTML and CSS
HTML is the skeleton for content. CSS is for styling and presentation. They are the two core languages of the web. 
<!-- HTML structures -->
<!-- CSS styles -->
 HTML provides the structure for a webpage, while CSS gives it a visual appearance.
Elements and Tags
An HTML **element** is the complete component. A **tag** is just the opening or closing part. 
<p>This is an element.</p>

This is an element.
HTML Boilerplate
The fundamental structure that every HTML page should have. It includes the `DOCTYPE`, `head`, and `body`. 
<!DOCTYPE html>
                        <html>
                        <head>
                            <title>Page Title</title>
                        </head>
                        <body>
                            <!-- Content goes here -->
                        </body>
                        </html>
 This code defines the basic structure that is required for a valid HTML document.
Working with Text
Tags for headings, paragraphs, and emphasizing text. 
<h1>Main Heading</h1>

Links and Images
Links (`<a>`) connect pages. Images (`<img>`) embed graphics. 
<a href="#">A link</a>
<img src="https://placehold.co/100x50" alt="A small box">
